```java
private void registerHard(final MOGroup group) {
    try {
        final Field registry = server.getClass().getDeclaredField("registry");
        registry.setAccessible(true);
        @SuppressWarnings("unchecked")
        SortedMap<MOScope, ManagedObject<ManagedObject>> reg = (SortedMap<MOScope, ManagedObject<ManagedObject>>) server.getRegistry();
        DefaultMOContextScope contextScope = new DefaultMOContextScope(new OctetString(""), group.getScope());
        reg.put(contextScope, group);
        registry.set(server, reg);
    } catch (NoSuchFieldException | ClassCastException | IllegalAccessException e) {
        log.warn("could not set server registry", e);
    }
}
```

The patch changes the type of the `reg` variable and adds a `@SuppressWarnings` annotation to suppress the unchecked cast warning. Also, the `ClassCastException` is added to the `catch` block to handle the potential `ClassCastException` during the unchecked cast.
