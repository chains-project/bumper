The error was caused by an incompatible type in the SortedMap, changing `SortedMap<MOScope, ManagedObject<?>>` to `SortedMap<MOScope, ManagedObject>` fixes the issue.

In addition, the method signature does not need to be changed, so the override decorator is not needed.
